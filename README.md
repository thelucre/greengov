### Install
Run:
* `bower install`
* `npm install`
* `webpack -w`

### Branches
The `master` branch ignores webpack builds. Change to the `gh-pages` branch and commit the webpack build js and css.

View the app at [http://thelucre.github.io/greengov](http://thelucre.github.io/greengov)
